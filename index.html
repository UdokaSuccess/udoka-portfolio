Where it all began
2022
A physiotherapy student who taught herself to code during a strike.

University went on strike. The country paused. Most students waited. I opened a laptop and started learning to build things on the internet. That is the origin. No bootcamp, no tech background, no one in my immediate circle who could point me toward a GitHub repository. Just a strike period, an internet connection, and the stubborn suspicion that software was something I could learn.

"The story I use to describe myself is the most honest one: I built these skills while earning a physiotherapy degree, alone, during a national pause."

I finished the degree. I also finished learning React, Node, Express, and MongoDB. Not because I had to pivot away from healthcare — but because I became someone who could see both worlds at once. A developer who understood clinical workflows. A builder who knew what it actually meant when a system failed someone in pain.

That dual perspective is not incidental. It is the engine behind Emergent, my healthcare tech startup. It is why I can write technical documentation that a nurse can read without a translator. It is why the fiction I write has structure, and why the code I write has intention.

Process

How I work.

Coding Workflow
01
Understand the problem before touching code
I ask: who is this for, what breaks without it, what does success look like in plain language.
02
Architecture — big picture first
Data model, component tree, API shape. I map it before I write the first line.
Excalidraw Notion
03
Build in layers — scaffold, fill, polish
Get it working ugly. Then make it clean. Then make it fast. In that order.
VS Code React Node
04
Test like a suspicious user
I try to break what I build. Edge cases, bad inputs, slow connections.
05
Ship, observe, iterate
Deployment is the beginning of learning, not the end of building.
Vercel Render GitHub
✎
Writing Workflow
01
Find the real question
Every piece of writing has a question underneath it. I don't start until I can name it.
02
Structure before prose
Outline with intention. Know where each section is going before writing the first sentence.
Notion iA Writer
03
Write fast, edit slow
First drafts are for thinking. Editing is where the writing actually happens.
04
Read it out loud
If I can't read it without pausing, it needs rewriting. Rhythm is non-negotiable.
05
Cut anything that doesn't earn its place
Good writing is mostly deletion. Every word should be doing something.

The why

Nigeria's emergency health system has a documentation and information-sharing problem. Patients arrive at hospitals with no transferable record. Clinicians make decisions in the dark. Emergent is being built to change that — one hospital at a time, starting in Enugu.

This is not a side project. I co-founded it. I lead the team. I am the developer, the project manager, and the person who stays up thinking about it. It is being kept private until hospital partnerships are secured.
